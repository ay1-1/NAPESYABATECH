# NAPES Yabatech | Site Map & Information Architecture

## 1. Public Facing (The Hub)
- **Home (Index)**
  - Hero: Mission Statement & Institutional Branding
  - Live Stats: Members, Resources, Projects Counter
  - Faculty/Departments: Overview of Engineering Schools
  - Latest Publications: News, Updates, Social proof
  - HerEngineering: Legacy program highlights
  - Industry Partners: Corporate logos and networking links

## 2. The Vault (Academic Resource Library)
- **Resource Categories**
  - Civil Engineering Repository
  - Mechanical Engineering Repository
  - Electrical/Electronics Repository
  - Computer Engineering Repository
  - Chemical Engineering Repository
- **Search & Filter**
  - Past Projects Search
  - Technical Paper Downloads
  - Tutorial Video Library

## 3. The Ledger (Student Portal)
- **Authentication**
  - Student Registration (Matric Number Verification)
  - Google Login Integration
- **Dashboard**
  - Profile Management
  - Dues Payment Status (Integrated with Paystack/Flutterwave)
  - Certificate Downloads (Dues clearance)
  - Election Voting Module (When active)

## 4. Governance & Community
- **Executive Council**
  - Profiles of current executives
  - Manifestos & Achievement reports
- **HerEngineering Programme**
  - Mentorship signup
  - Workshop schedules
- **Events**
  - Engineering Week coverage
  - Technical seminars

---

## Technical Stack
- **Frontend**: Next.js/React + Tailwind CSS
- **Animations**: GSAP (GreenSock) for high-end transitions + AOS for reveals
- **Backend**: Express.js (on Node)
- **Database**: Firebase Firestore
- **Auth**: Firebase Authentication
- **Payments**: Paystack API (Placeholder implemented)
